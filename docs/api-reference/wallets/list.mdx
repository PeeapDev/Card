# List Wallets

Retrieve all wallets belonging to the authenticated user.

## Endpoint

```
GET /wallets
```

## Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | Bearer access token |

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page` | integer | No | Page number (default: 1) |
| `limit` | integer | No | Items per page (default: 20, max: 100) |
| `currency` | string | No | Filter by currency code |
| `status` | string | No | Filter by status (ACTIVE, FROZEN, CLOSED) |

## Example Request

```bash
curl https://api.paymentsystem.com/v1/wallets \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### With Filters

```bash
curl "https://api.paymentsystem.com/v1/wallets?currency=USD&status=ACTIVE" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
