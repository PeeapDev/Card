# Students API

Sync your student database with Peeap to enable cashless payments and wallet linking.

## Sync Students

Bulk sync students from your school system to Peeap. This creates or updates student records and optionally links them to Peeap wallets.

<CodeGroup>
```bash cURL
curl -X POST "https://api.peeap.com/v1/schools/{schoolId}/students/sync" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "students": [
      {
        "studentId": "STU-2024-001",
        "firstName": "John",
        "lastName": "Doe",
        "email": "john.doe@school.edu",
        "phone": "+23276123456",
        "parentPhone": "+23277654321",
        "class": "Grade 10A",
        "admissionDate": "2024-01-15"
      }
    ]
  }'
```

```javascript JavaScript
const response = await fetch(`https://api.peeap.com/v1/schools/${schoolId}/students/sync`, {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${apiKey}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    students: [
      {
        studentId: 'STU-2024-001',
        firstName: 'John',
        lastName: 'Doe',
        email: 'john.doe@school.edu',
        phone: '+23276123456',
        parentPhone: '+23277654321',
        class: 'Grade 10A',
        admissionDate: '2024-01-15'
      }
    ]
  })
});
```
</CodeGroup>

### Response

```json
{
  "success": true,
  "data": {
    "synced": 1,
    "created": 1,
    "updated": 0,
    "failed": 0,
    "students": [
      {
        "studentId": "STU-2024-001",
        "peeapId": "usr_abc123",
        "walletId": "wal_xyz789",
        "status": "created",
        "walletLinked": true
      }
    ]
  }
}
```

## List Students

Get all students linked to your school.

```bash
GET /api/schools/{schoolId}/students
```

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `page` | integer | Page number (default: 1) |
| `limit` | integer | Items per page (default: 50, max: 200) |
| `status` | string | Filter: `linked`, `pending`, `unlinked` |
| `class` | string | Filter by class name |
| `search` | string | Search by name, ID, or phone |

{

## Link Student to Peeap Wallet

Link a student to an existing Peeap account by phone number.

```bash
POST /api/schools/{schoolId}/students/{studentId}/link
```

### Request Body

```json
{
  "phone": "+23276123456"
}
```

The system will search for a Peeap user with this phone number and link them to the student record.

### Response

```json
{
  "success": true,
  "data": {
    "studentId": "STU-2024-001",
    "peeapId": "usr_abc123",
    "walletId": "wal_xyz789",
    "linked": true,
    "message": "Student linked to Peeap wallet successfully"
  }
}
```

## Set Payment PIN

Set a 4-digit PIN for the student to authorize payments.

```bash
POST /api/schools/{schoolId}/students/{studentId}/set-pin
```

### Request Body

```json
{
  "pin": "1234"
}
```

<Warning>
The PIN is hashed and cannot be retrieved. If forgotten, it must be reset.
</Warning>
