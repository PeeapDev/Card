# List Cards

Retrieve all cards belonging to the authenticated user.

## Endpoint

```
GET /cards
```

## Headers

| Header | Required | Description |
|--------|----------|-------------|
| `Authorization` | Yes | Bearer access token |

## Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page` | integer | No | Page number (default: 1) |
| `limit` | integer | No | Items per page (default: 20, max: 100) |
| `walletId` | string | No | Filter by wallet ID |
| `status` | string | No | Filter by status |
| `type` | string | No | Filter by type (VIRTUAL, PHYSICAL) |

## Example Request

```bash
curl https://api.paymentsystem.com/v1/cards \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### With Filters

```bash
curl "https://api.paymentsystem.com/v1/cards?type=VIRTUAL&status=ACTIVE" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### Errors

| Status | Code | Description |
|--------|------|-------------|
| 401 | `UNAUTHORIZED` | Invalid or missing access token |

## Card Statuses

| Status | Description |
|--------|-------------|
| `PENDING_ACTIVATION` | Card created but not yet activated |
| `ACTIVE` | Card can be used for transactions |
| `BLOCKED` | Temporarily disabled |
| `EXPIRED` | Card has passed expiry date |
| `CANCELLED` | Permanently cancelled |
