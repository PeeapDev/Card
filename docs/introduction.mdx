# Welcome to Payment System

Payment System is a comprehensive closed-loop card payment platform that enables businesses to issue virtual and physical cards, manage wallets, and process transactions securely.

## Architecture Overview

Payment System is built on a microservices architecture:

- **Identity Service**: User authentication, registration, and KYC management
- **Account Service**: Wallet management and balance operations
- **Card Service**: Card issuance, activation, and management
- **Transaction Service**: Payment processing and settlement
- **API Gateway**: Unified entry point for all API requests
