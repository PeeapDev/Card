## Features

All SDKs provide:

- **Type safety**: Full TypeScript/type definitions
- **Authentication**: Automatic token management and refresh
- **Error handling**: Structured error responses
- **Retries**: Automatic retry with exponential backoff
- **Logging**: Configurable request/response logging

## Quick Comparison

## Common Patterns

### Authentication

All SDKs handle authentication automatically:

```javascript
// JavaScript
const client = new PaymentSystem({
  accessToken: 'your-access-token',
  refreshToken: 'your-refresh-token'
});
// Token refresh is handled automatically
```

```python
# Python
client = PaymentSystem(
    access_token='your-access-token',
    refresh_token='your-refresh-token'
)
# Token refresh is handled automatically
```

### Error Handling

```javascript
// JavaScript
try {
  const wallet = await client.wallets.create({ currencyCode: 'USD' });
} catch (error) {
  if (error instanceof PaymentSystemError) {
    console.error(error.code, error.message);
  }
}
```

```python
# Python
try:
    wallet = client.wallets.create(currency_code='USD')
except PaymentSystemError as error:
    print(error.code, error.message)
```
