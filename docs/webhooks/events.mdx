# Webhook Events

Complete reference for all webhook event types.

## Transaction Events

### transaction.authorized

Sent when a transaction is authorized (funds held).

```json
{
  "id": "evt_abc123",
  "type": "transaction.authorized",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "txn_xyz789",
    "type": "AUTHORIZATION",
    "status": "AUTHORIZED",
    "amount": 99.99,
    "currency": "USD",
    "cardId": "crd_def456",
    "walletId": "wal_ghi789",
    "merchant": {
      "name": "Example Store",
      "category": "5411",
      "city": "New York",
      "country": "US"
    },
    "authorizationCode": "AUTH123"
  }
}
```

### transaction.completed

```json
{
  "id": "evt_abc124",
  "type": "transaction.completed",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "txn_xyz789",
    "type": "CAPTURE",
    "status": "CAPTURED",
    "amount": 99.99,
    "currency": "USD",
    "cardId": "crd_def456",
    "walletId": "wal_ghi789",
    "merchant": {
      "name": "Example Store",
      "category": "5411"
    }
  }
}
```

### transaction.declined

Sent when a transaction is declined.

```json
{
  "id": "evt_abc125",
  "type": "transaction.declined",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "txn_xyz790",
    "type": "PURCHASE",
    "status": "DECLINED",
    "amount": 500.00,
    "currency": "USD",
    "cardId": "crd_def456",
    "declineReason": "INSUFFICIENT_FUNDS",
    "responseCode": "51"
  }
}
```

### transaction.refunded

Sent when a transaction is refunded.

```json
{
  "id": "evt_abc126",
  "type": "transaction.refunded",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "txn_refund_abc",
    "originalTransactionId": "txn_xyz789",
    "type": "REFUND",
    "status": "COMPLETED",
    "amount": 25.00,
    "currency": "USD",
    "reason": "Partial return"
  }
}
```

### card.created

Sent when a new card is issued.

```json

  "type": "card.created",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "crd_new123",
    "type": "VIRTUAL",
    "status": "PENDING_ACTIVATION",
    "lastFour": "4242",
    "walletId": "wal_abc123",
    "userId": "usr_def456"
  }
}
```

### card.activated

```json

  "type": "card.activated",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "crd_new123",
    "status": "ACTIVE",
    "activatedAt": "2024-01-15T10:30:00Z"
  }
}
```

### card.blocked


## Wallet Events

### wallet.created

```json

  "id": "evt_wal001",
  "type": "wallet.created",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {

    "currencyCode": "USD",
    "status": "ACTIVE",
    "ownerId": "usr_abc123",
    "ownerType": "USER"
  }
}
```

### wallet.frozen

Sent when a wallet is frozen.

```json
{
  "id": "evt_wal002",
  "type": "wallet.frozen",
  "createdAt": "2024-01-15T10:30:00Z",
```

```json
{
  "id": "evt_wal003",
  "type": "wallet.low_balance",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "id": "wal_abc123",
    "currencyCode": "USD",
    "availableBalance": 25.00,
    "threshold": 50.00
  }
}
```

## KYC Events

### kyc.submitted

Sent when KYC documents are submitted.

```json

  "type": "kyc.submitted",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "userId": "usr_abc123",
    "status": "PENDING",
    "submittedAt": "2024-01-15T10:30:00Z",
    "requestedTier": 2
  }
}
```

### kyc.approved

Sent when KYC verification is approved.

```json
{

  "createdAt": "2024-01-15T10:30:00Z",
  "data": {

```

```json

  "type": "kyc.rejected",
  "createdAt": "2024-01-15T10:30:00Z",
  "data": {
    "userId": "usr_abc123",

    "rejectedAt": "2024-01-15T10:30:00Z",
    "reasons": ["DOC_UNREADABLE", "ADDRESS_INVALID"]
  }
}
```

## Subscribing to Events

When registering a webhook, specify which events to receive:

```bash
curl -X POST https://api.paymentsystem.com/v1/webhooks \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-server.com/webhooks",
    "events": [
      "transaction.completed",
      "transaction.declined",
      "card.blocked",
      "wallet.low_balance"
    ]
  }'
```

Use `"events": ["*"]` to subscribe to all events.
